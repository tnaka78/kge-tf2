# KGE-TF2
Knowledge graph embeding implementation with Tensorflow 2
 * TransE ([Translating Embeddings for Modeling Multi-relational Data](https://www.utc.fr/~bordesan/dokuwiki/_media/en/transe_nips13.pdf))
## Requirements
 * Python >= 3.6
 * Tensorflow >= 2.1.0

## How to Run
```
python main.py
```

## Dataset
Data are from [here](https://github.com/thunlp/OpenKE)
